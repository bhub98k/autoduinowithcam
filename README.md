本專案旨在開發一套智慧自動追蹤攝影系統，整合 Arduino UNO、Raspberry Pi、伺服馬達與 USB Camera，實現即時目標追蹤與自動構圖調整。系統可偵測講者位置，透過伺服馬達控制攝影機水平與垂直轉動，使講者始終位於畫面中央。
本設計聚焦於解決教室錄影場景中「黑板範圍過寬、攝影機視角受限」的實務問題。透過機電整合與影像辨識技術，系統可動態調整拍攝角度，在有限視角下維持最佳構圖品質，提升課程錄製的專業度與觀看體驗。
核心特色包含：
即時影像擷取與目標追蹤
伺服馬達精準角度控制
Raspberry Pi 與 Arduino 分工協作架構
可擴充之影像辨識與自動化控制模組
本專案適用於智慧教室、自動錄播系統與低成本自動攝影應用場景，並可作為影像辨識與機電整合實作之教學與研究範例。
